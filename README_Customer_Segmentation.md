# Customer Segmentation using K-Means

## Project Overview
This project performs **Customer Segmentation** using the **K-Means Clustering** algorithm.
The objective is to group customers based on purchasing behaviour and demographic features to help businesses create targeted marketing strategies.

## Dataset
- 12,000+ Records
- Features:
  - CustomerID
  - Age
  - Annual Income
  - Spending Score
  - Gender

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Workflow
1. Load Dataset
2. Data Cleaning & EDA
3. Feature Scaling
4. Elbow Method
5. K-Means Clustering
6. Cluster Visualization
7. Cluster Profiling

## Key Insights
- High-income, high-spending customers can be targeted with premium campaigns.
- Low-spending customers may respond to discounts and promotions.
